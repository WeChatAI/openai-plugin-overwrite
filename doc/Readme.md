# 微信对话开放平台小程序插件复写示例
微信对话开放平台小程序插件复写示例提供了对插件的开放能力， 可以随意定制化。

![链接](./one.jpg)

![链接](./two.jpg)

## 1.普通文本组件复写示例

```json

{
  "usingComponents": {
    "customTextMessage": "../customTextMessage/customTextMessage"
  }
}

```
